2022.04.05

关于我的 Minecraft 服务器。

前言

这篇文章是搭建联机用Minecraft服务器的教学。这里是指可以用来与好友联机的服务器。

搭建之前，你得准备这几样东西。（以运行1.18.0并最高人数10人的流畅标准为准）

  • 一台台式电脑(带有靠谱电源)、或者是1核2G的云服务器

    台式电脑的建议配置 Intel 3代 四核、不需要显卡、Windows Server2019 x64或Win10 x64专业版的电脑（便于操作）

    云服务器的建议配置 1核2G 3~5M带宽 详见：百度知道大佬们的回答 并安装上示系统

  • SakuraFrp（是目前来说带宽最高，福利最优的内网穿透）拥有公网IP的云服务器可以忽略

  • Java或者JDK11和一个Minecraft核心，（从1.18开始，你应该安装JDK17或更高）

  • 其他没了

搭建

配置服务端

先下载Java或者JDK11（从1.18开始，你应该安装JDK17或更高）

然后再下载Minecraft核心。这里给出McBBS的帖子，请自行下载。这是帖子(仔细阅读帖子内的服务端介绍格式)

接着，配置服务器的启动脚本。请在储存Minecraft核心的文件夹下，建立一个文本文档，然后输入：

java -server -Xmx游戏最大内存M -Xms游戏最小内存M -jar 服务端的文件名.jar nogui

可以在后面再补上一行pause，这样就不至于让游戏出错后一闪而过

比如

java -server -Xmx1024M -Xms1024M -jar mcserver.jar nogui

pause

然后另存为.Bat格式的文件。

运行这个文件，会出现一只黑框。黑框里会出现如下内容，

[hh:mm:ss] [main/WARN]: Failed to load eula.txt
[hh:mm:ss] [main/INFO]: You need to agree to the EULA in order to run the server. Go to eula.txt for more info.

你需要打开目录下的eula文件（Mojang的最终许可协议），仔细阅读https://account.mojang.com/documents/minecraft_eula 然后将eula=false改为elua=true

内网穿透

前往SakuraFrp，注册一个账户，然后创建一个隧道

详细设置如下图

请注意，“本地端口”填写25565

如果您的服务器经常被 DOS 攻击，请使用高防节点 (如枣庄)

然后下载SakuraFrp的客户端，填入在主页获取到的访问密钥，然后点开对应隧道的开关即可。

如果在这个过程遇到问题，你应该在帮助文档 - Sakura Frp (natfrp.com)中查询答案。篇幅所限，不做赘述。

正式启动

打开内网穿透工具，启用隧道，然后打开脚本即可

遇到问题？

客户端

连接中止

你的MC服务端可能崩溃。

Connection reset

你选择了被墙的国外通道，或者是不稳定的国内通道。如果是隧道被墙，你应该更换隧道。如果隧道不稳定，你可以试着多连几次。

Connection refused: No further information

  1. 服务器后台崩溃或者关闭。
  2. 服务器已满。
  3. 你打开端口映射了吗？
  4. 服务器的插件有冲突，导致服务器后台崩溃。
  5. 客户端的网速太慢了，或者服务器的带宽太逊了

登入失败：无效的会话（请尝试重启你的游戏）

你开启了正版验证，反映此错误的玩家可能是没有正版账号的玩家。

如果你要关闭正版验证，请打开服务器核心目录下的server.properties。然后将online-mode的值改为false

远程主机强迫关闭了一个现有的连接

  1. 服务器网络不好，或者客户端的网速满。
  2. 客户端安装了MOD，但是与服务器冲突了。
  3. 服务器要求客户端安装MOD，但是客户端并没有这种MOD。常见于MOD生存，比如神奇宝贝服务器
  4. 内网穿透正在检修
  5. 反作弊侦测到了外挂，或者是误判。

Unknow Hostname

你绑定的域名无效。请注意CNAME解析对应着域名，A解析对应着IP。

服务端

Can’t keep up! Did the system time change, or is the server overloaded?

  1. 侦测到系统时间被改变
  2. CPU占用率太高了。请检查。
  3. 服务器使用了太多的插件。如果你是小众生存服务器，只需安装反作弊，锁箱子，圈地即可
